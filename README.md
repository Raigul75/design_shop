# Design Shop

Проект полностью готов к статическому деплою и содержит встроенный локальный сервер для проверки.

## Запуск проекта локально (Ручной режим)

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Установите зависимости (Express):
   ```bash
   npm install
   ```
3. Запустите локальный сервер:
   ```bash
   npm start
   ```
4. Откройте в браузере `http://localhost:3000` (если порт 3000 занят, сервер использует доступный, либо можно задать свой через `PORT=8080 npm start`).

## Деплой в интернет (Автоматический через Netlify)

Проект содержит конфигурационный файл `netlify.toml`, который позволяет развернуть сайт в один клик.

### Вариант 1: Через GitHub (Рекомендуемый автоматический метод)
1. Убедитесь, что весь код запушен в ваш репозиторий GitHub.
2. Зарегистрируйтесь на [Netlify](https://www.netlify.com/).
3. Нажмите **Add new site** -> **Import an existing project**.
4. Выберите **GitHub** и найдите репозиторий `design_shop`.
5. Netlify автоматически подхватит настройки из `netlify.toml`.
6. Нажмите **Deploy site**.
7. Теперь любое ваше изменение и `git push` в ветку `main` будет автоматически обновлять сайт!

### Вариант 2: Ручной деплой (Drag & Drop)
1. Зайдите на [Netlify Drop](https://app.netlify.com/drop).
2. Перетащите папку с проектом в специальное окно на сайте.
3. Ваш сайт будет опубликован мгновенно.

## Архитектура
Проект использует классический стек HTML/CSS/JS. Настроен Express-сервер (`server.js`) с поддержкой динамического порта (`process.env.PORT || 3000`), что позволяет деплоить его не только на статические хостинги, но и на облачные платформы вроде Heroku или Render.
